## Getting Started
1. Clone this repo
```git
git clone https://github.com/ChihuahuaMH/egg_blog.git
```

2. Install NPM packages to `client` & `server` folder individually.
```
npm install
```

3. Run dev
```
cd ./server & npm run dev
```
